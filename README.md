# Soft_NeuroKit_nRF
This application uses SPI to send EEG signals from ADS1299 to nRF52840 and BLE to send digital EEG signals to the mobile app

Setup before git pull: https://www.youtube.com/watch?v=EAJdOqsL9m8

VsCode -> Open Nordic_BLE_EEG -> Add build configuration -> select nRF52840 & prj.conf -> Build all -> Flash all
